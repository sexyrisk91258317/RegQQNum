首先，打开QQ官网或者QQ手机客户端，点击“注册”按钮。然后根据页面提示填写个人信息，包括手机号码、密码、昵称等。在填写完毕后，点击“获取验证码”并输入收到的验证码，然后点击“注册”按钮即可完成注册。
